<div align="center">
    <h2>go-commit</h2>
    <p>The CLI to generate commit with karma style</p>
</div>

### Prerequisite

- [Go](https://go.dev/doc/install)
- [OpenAI Key](https://platform.openai.com/account/api-keys)

### Install

```bash
go install github.com/ak9024/go-commit@latest
```

### How to run?

```bash
cd <your-repo>
git add .
go-commit
```
